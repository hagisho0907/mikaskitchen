# Mika-s-Kitchen
Mika's Kitchen HP

# 概要
”美加の台所”という自然食品を扱う、古民家を利用した飲食店のHP

# 参考
https://green-dish.com/

# 技術要素
フロント：Next.js
UIコンポーネントライブラリ：Chakra UI + Motion
デプロイ先：Vercel
